이 프로젝트는 대시보드 스타터킷에서 Notion CMS 기반 개인 기술 블로그로 전환됨.

**Why:** PRD 요구사항 — Notion을 CMS로 활용하여 글쓰기 효율성 증대, 별도 관리자 페이지 없이 Notion 인터페이스로 콘텐츠 관리.

**How to apply:** 이후 작업 시 (auth), (dashboard), (marketing) route group은 존재하지 않음. 블로그 구조는 (blog) route group 기준으로 설계.
